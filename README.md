# Qual o CEP

Qual o CEP é um aplicativo mobile desenvolvido React Native que tem como objetivo saber o CEP de qualquer cidade do Brasil.

## Instalação

Use o gerenciador de pacotes `YARN`

```bash
pip install foobar
```

## Usage

```python
import foobar

foobar.pluralize('word') # returns 'words'
foobar.pluralize('goose') # returns 'geese'
foobar.singularize('phenomena') # returns 'phenomenon'
```
